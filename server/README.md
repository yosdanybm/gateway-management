## Installation packages

```bash
$ npm install
```

## Configure the connection with your MongoDB database 

```bash
# app.module.ts
'mongodb://user:password@localhost:27017/name_your_database'
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
